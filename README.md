Na segunda fase deste projeto, a missão era criar um código que filtrasse os dados fornecidos pela empresa de jogos e respondesse 8 perguntas pertinentes aos dados, além da criação de 3 gráficos.


O .zip com os dados pode ser encontrado no repositório da primeira fase do projeto: https://github.com/henriquepetiz/game-data-analysis-p1

Foram criadas 2 versões de código para as mesmas perguntas, embora 'main_updated.py' ser o código criado para ser mais simples e rápido no processamento, 'main.py' foi o código entregue junto ao projeto, sendo ele o qual eu reconheci como mais satisfatório.
